## 📋 PRÉ-REQUISITOS

1. **Python 3.8+** instalado
2. **GitHub Token** (Personal Access Token)
3. **pip** configurado

### Gerando um GitHub Token

1. Acesse: https://github.com/settings/tokens
2. Clique em "Generate new token" > "Generate new token (classic)"
3. Configure as permissões:
   - ✓ public_repo (acesso a repositórios públicos)
   - ✓ read:user (ler dados públicos do perfil)
   - ✓ repo (acesso geral a repositórios)
4. Copie o token gerado (não será exibido novamente)

## 🚀 PASSO A PASSO

### PASSO 1: Configurar Ambiente
```bash
# Na pasta enunciado03/

# 1.1 Copiar arquivo .env
copy .env.example .env

# 1.2 Editar .env e substituir:
#     GITHUB_TOKEN=seu_token_aqui_sem_aspas
# Com seu token real gerado acima

# 1.3 Instalar dependências
pip install -r requirements.txt
```

### PASSO 2: Testar Conexão com GitHub
```bash
# Verificar se a conexão está funcionando
python src/github_query.py
```

Você deve ver uma mensagem como:
```
✓ Conectado ao GitHub API
✓ Rate limit: 4950/5000
✓ Configuração validada com sucesso!
```

### PASSO 3: Selecionar Repositórios
```bash
# Coletar os 200 repositórios mais populares
python src/repository_selector.py
```

Isto irá:
- Buscar repositórios ordenados por número de stars
- Filtrar apenas aqueles com ≥ 100 PRs (MERGED + CLOSED)
- Salvar em: enunciado03/data/selected_repositories.csv

Tempo esperado: ~10-12 minutos ⚡ (otimizado)

### PASSO 4: Coletar PRs e Métricas
```bash
# Coletar PRs e calcular todas as métricas
python src/pr_metrics.py
```

Isto irá:
- Para cada repositório selecionado, buscar PRs com os critérios:
  ✓ Status: MERGED ou CLOSED
  ✓ Mínimo 1 revisão
  ✓ Tempo de revisão > 1 hora
- Salvar em: enunciado03/data/pr_metrics.csv

Tempo esperado: ~30-40 minutos ⚡ (otimizado, era 30-60 min)

### PASSO 5: Executar Tudo de Uma Vez
```bash
# Executar todos os passos automaticamente
python src/main.py
```

Isto irá:
1. Selecionar repositórios
2. Coletar PRs e métricas
3. Gerar relatório inicial
4. Salvar tudo em arquivos CSV

Tempo esperado: ~45 minutos ⚡ (otimizado, era 1-2 horas)
